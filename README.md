# Books Mori Library

A cozy library app built with HTML, CSS and JavaScript. 

Users can add books with title, author, year, pages and reading status. 
This project helped me practice DOM manipulation, form handling, JavaScript objects and basic UI design. 

## Features

- Add a book using a popup form
- Display book information
- Delete book cards
- Track reading status
- Cozy responsive design

## Built With
- HTML
- CSS
- JavaScript

# What I Practiced

- DOM manipulation
- Event listeners
- Form handling
- Constructor functions / objects
- Working with arrays
- Basic UI styling
